# House Price Prediction using Machine Learning

## Objective
To predict house prices using regression models based on property features.

## Dataset
House Prices – Advanced Regression Techniques (Kaggle)

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Models Implemented
- Linear Regression (R² ≈ 0.76)
- Random Forest Regressor (Improved performance)

## Evaluation Metrics
- RMSE
- R² Score

## Conclusion
Random Forest outperformed Linear Regression by capturing non-linear relationships in the data.
